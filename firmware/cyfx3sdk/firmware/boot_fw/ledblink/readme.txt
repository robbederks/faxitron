
                                CYPRESS SEMICONDUCTOR

                                   EZ-USB FX3 SDK

                                 BootLedBlink example
                                 --------------------

  This directory contains an example application that shows the usage of the FX3 boot API
  to sample Switches and control GPIOs.

  The application senses the state of the GPIO45 pin on the FX3 part and drives the same value
  out on the GPIO54 pin. If the Cypress SuperSpeed Explorer Kit is used, the SW2 switch can be
  used to control the state of the LED2 pin.

  Files:

  1. cyfx_gcc_startup.S  : Start-up code for the FX3 device.

  2. defines.h           : Definitions used in the application code.

  3. main.c              : Application source code.

  4. makefile            : GNU make compatible build recipe.

  5. readme.txt          : This file.
